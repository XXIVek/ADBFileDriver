# Драйвер ADB для 1С:Предприятие 8.3

## Общая информация

**Цель:** Создание внешней компоненты для передачи информации между 1С:Предприятие 8.3 и Android устройствами через прямой USB доступ.

---

## История изменений

### Версия 1.0.0.1 (01.07.2026)
- Создана базовая структура проекта
- Реализована MVP версия компоненты для Win64
- Добавлено одно свойство `Версия` для проверки подключения

## Сборка проекта

### Требования
- CMake 3.10+
- MSVC (Visual Studio 2019 или новее)

### Команда сборки
```powershell
# Создание директории build
New-Item -ItemType Directory -Path build -Force
cd build

# Генерация проекта
cmake .. -G "Visual Studio 17 2022" -A x64

# Сборка
cmake --build . --config Release
```

### Результат
После сборки DLL будет расположена в:
```
J:\Visual studio prog\ADBFileDriver\1C\Release\Win64\ADBFileDriver_Win64.dll
```

## Подключение компоненты в 1С

### Способ 1: Подключение из файла на диске

```bsl
// Подключение компоненты
ИмяКомпоненты = "ADBFileDriver";
ПутьКФайлу = "C:\ADBFileDriver\1C\Release\Win64\ADBFileDriver_Win64.dll";

Результат = ПодключитьВнешнююКомпоненту(ПутьКФайлу, ИмяКомпоненты, ТипВнешнейКомпоненты.Native);

Если Результат Тогда
    // Создание объекта компоненты
    Драйвер = Новый("AddIn.ADBFileDriver.ADBFileDriver");
    
    // Проверка версии
    Сообщить(Драйвер.Версия); // "1.0.0.1"
КонецЕсли;
```

### Способ 2: Подключение из макета конфигурации

```bsl
// Получение макета компоненты из конфигурации
ДвоичныеДанные = ПолучитьМакет("ВнешняяКомпонентаADBFileDriver");

// Подключение из двоичных данных
Результат = ПодключитьВнешнююКомпоненту(
    ДвоичныеДанные,
    "ADBFileDriver",
    ТипВнешнейКомпоненты.Native
);

Если Результат Тогда
    Драйвер = Новый("AddIn.ADBFileDriver.ADBFileDriver");
    Сообщить(Драйвер.Версия);
КонецЕсли;
```

### Способ 3: Подключение из ZIP-архива

```bsl
// Создание ZIP-архива с компонентой
// Внутри архива должна быть структура:
//   bundle/
//     MANIFEST.xml
//   Win64/
//     ADBFileDriver_Win64.dll

ПутьКАрхиву = "C:\ADBFileDriver\ADBFileDriver.zip";

Результат = ПодключитьВнешнююКомпоненту(
    ПутьКАрхиву,
    "ADBFileDriver",
    ТипВнешнейКомпоненты.Native
);
```

## Требования к среде выполнения

1. **Операционная система:** Windows 10/11 (64-bit)
2. **USB драйверы:** Android USB драйвер должен быть установлен на устройстве
3. **ADB Debugging:** На Android устройстве должна быть включена "Отладка по USB"
