# Драйвер ADB для 1С:Предприятие 8.3

## Описание

Драйвер обеспечивает работу с Android-устройствами через ADB (Android Debug Bridge) для файловой работы.

## Возможности

- Перечисление подключенных USB- и сетевых устройств (эмуляторов)
- Подключение/отключение устройств
- Работа с файлами на устройстве (список, загрузка, выгрузка, удаление)

## Поддерживаемые устройства

- Физические Android-устройства через USB
- Эмуляторы с префиксом "emulator-" в серийном номере
- Сетевые устройства (через tcpip)

## Логирование

### Включение логирования

```bsl
Драйвер.ВключитьЛогирование = Истина;
Драйвер.ПутьДляФайлаЛогирования = "C:\temp\ADBFileDriver.log";
```

### Формат лог-файла

Файл создаётся в кодировке UTF-8 с BOM. Каждая строка имеет формат:

```
[ЧЧ:ММ:СС.ммм] ТЕГ: сообщение
```

### Форматы записей

#### 1. Подключение компоненты
```
[INIT] Факт подключения компоненты: успешно
```

#### 2. Состояние свойств
```
[SET] Свойство EnableLog = true
[SET] Свойство LogPath = C:\temp\ADBFileDriver.log
[SET] Свойство КаталогADB = C:\temp\ADB\
```

#### 3. Выполнение метода ПеречислитьУстройства()
```
[METHOD] Выполнение метода ПеречислитьУстройства()
[METHOD] ПеречислитьУстройства() результат=[{"Serial":"X4DENVHMJ7M7RWAE","Name":"M2006C3MNG"},{"Serial":"emulator-5554","Name":"sdk_gphone_x86"}]
```

#### 4. Выполнение метода Подключить(ИмяУстройства)
```
[METHOD] Выполнение метода Подключить(emulator-5554)
[METHOD] Подключить(emulator-5554) результат=Успех, подключено к: emulator-5554
```

#### 5. Свойства Status и DeviceCount
```
Status = "Подключено (ADB)"
DeviceCount = 2
```

#### 6. Выполнение метода СписокФайлов(Путь)
```
[METHOD] Выполнение метода СписокФайлов(/sdcard/Download)
[METHOD] СписокФайлов(/sdcard/Download) результат=Успех, FileCount=6, список=[{"Name":"Browser","IsFolder":true},{"Name":"LogADBFileDriver.log","IsFolder":false},...]
```

#### 7. Свойство FileCount
```

```

#### 8. Выполнение метода ЗагрузитьФайл(ИмяФайла, Содержание)
```
[METHOD] Выполнение метода ЗагрузитьФайл(myfile.txt)
[METHOD] ЗагрузитьФайл(myfile.txt) результат=Успех
```

#### 9. Выполнение метода СкачатьФайл(ИмяФайла)
```
[METHOD] Выполнение метода СкачатьФайл(myfile.txt)
[METHOD] СкачатьФайл(myfile.txt) результат=Успех
```

#### 10. Выполнение метода УдалитьФайл(ИмяФайла)
```
[METHOD] Выполнение метода УдалитьФайл(myfile.txt)
[METHOD] УдалитьФайл(myfile.txt) результат=Успех
```

#### 11. Отключение компоненты
```
[DONE] Факт отключения компоненты: начинается
[DONE] Факт отключения компоненты: завершено, ADB сервер не завершается (оставлен для других процессов)
```

## Пример использования в 1С

```bsl
// Подключение компоненты
Компонента = Новый("AddIn.ADBFileDriver.ADBFileDriver");

// Настройка логирования
Компонента.ВключитьЛогирование = Истина;
Компонента.ПутьДляФайлаЛогирования = "C:\temp\ADBFileDriver.log";

// Перечисление устройств
СписокУстройств = Компонента.ПеречислитьУстройства();

// Подключение к устройству
Результат = Компонента.Подключить("emulator-5554");

// Получение списка файлов
Список = Компонента.СписокФайлов("/sdcard/Download");

// Загрузка файла
Компонента.ЗагрузитьФайл("test.txt", "Содержимое файла");

// Скачивание файла
Содержимое = Компонента.СкачатьФайл("test.txt");

// Удаление файла
Компонента.УдалитьФайл("test.txt");

// Отключение
Компонента.Отключить();
```
