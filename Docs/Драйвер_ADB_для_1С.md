# Драйвер ADB для 1С:Предприятие 8.3

## Описание
Внешняя компонента для обмена данными между 1С:Предприятие 8.3 и Android устройствами через ADB (Android Debug Bridge).

## Установка
1. Скопируйте `ADBFileDriver_Win64.dll` в нужную директорию
2. Скопируйте `bundle/MANIFEST.xml` в папку `bundle/` рядом с DLL
3. Загрузите ADB драйверы через диспетчер устройств

## Подключение в 1С

```bsl
// Подключение компоненты
Результат = ПодключитьВнешнююКомпоненту(
    "J:\Visual studio prog\ADBFileDriver\1C\Release\Win64\ADBFileDriver_Win64.dll",
    "ADBFileDriver",
    ТипВнешнейКомпоненты.Native
);

Если Результат Тогда
    Драйвер = Новый("AddIn.ADBFileDriver.ADBFileDriver");
КонецЕсли;
```

## Свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| `Версия` | Строка | Номер версии компоненты ("1.0.0.1") |
| `ВключитьЛогирование` | Булево | true/false включение логирования |
| `ПутьДляФайлаЛогирования` | Строка | Путь к файлу лога (по умолчанию: %TEMP%\ADBFileDriver.log) |
| `Статус` | Строка | "Не подключен" / "Подключено" |
| `DeviceCount` | Число | Количество найденных USB устройств |

## Методы

### Подключить()
Подключает компоненту (в MVP - просто устанавливает статус).

**Возвращаемое значение:** Boolean - true при успехе

### Отключить()
Отключает компоненту.

**Возвращаемое значение:** Boolean - true при успехе

### ПеречислитьУстройства()
Перечисляет все подключенные Android устройства через ADB.

**Возвращаемое значение:** Строка (JSON формат)

**Пример JSON:**
```json
[
  {
    "serial": "RF8T5XXXXXXXX",
    "model": "Galaxy S21",
    "device": "s5e",
    "type": "usb",
    "vendorId": 0x04E8,
    "productId": 0x6860
  }
]
```

### ПолучитьИнформациюОУстройстве(Индекс)
Возвращает информацию об устройстве по индексу.

**Параметры:**
- `Индекс` (Число) - индекс устройства (0-based)

**Возвращаемое значение:** Строка (JSON формат с информацией об устройстве)

## Пример использования

```bsl
// Подключение
Результат = ПодключитьВнешнююКомпоненту(
    "путь/к/ADBFileDriver_Win64.dll",
    "ADBFileDriver",
    ТипВнешнейКомпоненты.Native
);

Если Результат Тогда
    Драйвер = Новый("AddIn.ADBFileDriver.ADBFileDriver");
    
    // Проверка версии
    Сообщить(Драйвер.Версия); // "1.0.0.1"
    
    // Включение логирования
    Драйвер.ВключитьЛогирование = Истина;
    Драйвер.ПутьДляФайлаЛогирования = "C:\Logs\ADBFileDriver.log";
    
    // Подключение
    Драйвер.Подключить();
    Сообщить(Драйвер.Статус); // "Подключено"
    
    // Перечисление устройств
    JSON = Драйвер.ПеречислитьУстройства();
    Сообщить(JSON);
    
    // Получение количества устройств
    Сообщить(Драйвер.DeviceCount);
    
    // Отключение
    Драйвер.Отключить();
    Сообщить(Драйвер.Статус); // "Не подключен"
    
    // Безопасное уничтожение
    Драйвер = Неопределено;
КонецЕсли;
```
