# Подробное описание получения списка файлов через ADBFileDriver в 1С

## Метод СписокФайлов(Путь)

### Назначение
Метод возвращает список файлов и папок на подключенном MTP устройстве в формате JSON.

### Синтаксис
```bsl
Список = Driver.СписокФайлов(Путь)
```

### Параметры
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| Путь | Строка | Нет | Относительный путь от корня устройства. Пустая строка или не передан — корневая директория. |

### Возвращаемое значение
Строка в формате JSON (массив объектов).

### Формат JSON
```json
[
  {
    "Id": "object_id",
    "Name": "имя_файла",
    "Size": 12345,
    "IsFolder": false,
    "MimeType": "image/jpeg"
  },
  {
    "Id": "object_id_2",
    "Name": "папка",
    "Size": 0,
    "IsFolder": true,
    "MimeType": "folder"
  }
]
```

### Поля объектов в JSON
| Поле | Тип | Описание |
|------|-----|----------|
| Id | Строка | Уникальный идентификатор объекта на устройстве |
| Name | Строка | Имя файла или папки |
| Size | Число | Размер в байтах (для папок = 0) |
| IsFolder | Булево | true — если объект является папкой, false — если файл |
| MimeType | Строка | MIME-тип файла (например, image/jpeg, application/pdf, folder) |

## Полный пример получения списка файлов

### Шаг 1: Создание компоненты

```bsl
// Создаем компоненту ADBFileDriver
Driver = New('AddIn.ADBFileDriver.ADBFileDriver');
```

### Шаг 2: Настройка логирования (опционально)

```bsl
// Логирование выключено по умолчанию.
// Включите только если нужна отладка:
// Driver.EnableLog = Истина;
// Driver.LogPath = "%TEMP%\ADBFileDriver.log";
```

### Шаг 3: Перечисление устройств

```bsl
// Получаем список подключенных MTP устройств
JSONСтрока = Driver.ПеречислитьУстройства();

// Читаем JSON результат
ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.УстановитьСтроку(JSONСтрока);
МассивУстройств = ПрочитатьJSON(ЧтениеJSON);

// Закрываем JSON
ЧтениеJSON.Закрыть();

// Выводим список устройств
Для каждого Устройство Из МассивУстройств Цикл
    Сообщить("Устройство: " + Устройство.Name + " [ID: " + Устройство.Id + "]");
КонецЦикла;

Сообщить("Всего устройств: " + Driver.DeviceCount);
```

### Шаг 4: Подключение к устройству

```bsl
// Подключаемся к первому найденному устройству
Если МассивУстройств.Количество() > 0 Тогда
    ПервоеУстройство = МассивУстройств[0];
    
    Если Driver.Подключить(ПервоеУстройство.Name) = Истина Тогда
        Сообщить("Успешно подключено к: " + Driver.Status);
    Иначе
        Сообщить("Ошибка подключения к устройству: " + Driver.Status);
        Возврат;
    КонецЕсли;
Иначе
    Сообщить("Подключенные устройства не найдены!");
    Возврат;
КонецЕсли;
```

### Шаг 5: Получение списка файлов в корне устройства

```bsl
// Получаем список файлов в корневой директории устройства
JSONФайлов = Driver.СписокФайлов("");

// Проверяем результат
Если Driver.FileCount = 0 Тогда
    Сообщить("Файлы не найдены");
Иначе
    Сообщить("Найдено файлов/папок: " + Driver.FileCount);
    
    // Парсим JSON результат
    ЧтениеJSON = Новый ЧтениеJSON;
    ЧтениеJSON.УстановитьСтроку(JSONФайлов);
    МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
    ЧтениеJSON.Закрыть();
    
    // Выводим список файлов
    Для каждого Файл Из МассивФайлов Цикл
        Если Файл.IsFolder = Истина Тогда
            ТипОбъекта = "[Папка]";
        Иначе
            ТипОбъекта = "[Файл]";
        КонецЕсли;
        
        РазмерСтрока = Если(Файл.Size > 0, 
            Формат(Файл.Size, "ЧЛЦ=12; ЧГ=0"), 
            "0");
        
        Сообщить(ТипОбъекта + " | " + Файл.Name + " | " + РазмерСтрока + " байт | " + Файл.MimeType);
    КонецЦикла;
КонецЕсли;
```

### Шаг 6: Получение списка файлов в конкретной папке

```bsl
// Получаем список файлов в папке InternalStorage/Downloads
JSONФайлов = Driver.СписокФайлов("/InternalStorage/Downloads");

Если Driver.FileCount > 0 Тогда
    ЧтениеJSON = Новый ЧтениеJSON;
    ЧтениеJSON.УстановитьСтроку(JSONФайлов);
    МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
    ЧтениеJSON.Закрыть();
    
    Для каждого Файл Из МассивФайлов Цикл
        Если Файл.IsFolder = Истина Тогда
            Сообщить("  [Папка] " + Файл.Name);
        Иначе
            Сообщить("  [Файл] " + Файл.Name + " (" + Формат(Файл.Size, "ЧЛЦ=12; ЧГ=0") + " байт)");
        КонецЕсли;
    КонецЦикла;
КонецЕсли;
```

### Шаг 7: Рекурсивный обход папок

```bsl
// Функция рекурсивного обхода папок
Функция ОбойтиПапку(Путь, Глубина)
    Префикс = "";
    Для г = 1 По Глубина Цикл
        Префикс = Префикс + "  ";
    КонецЦикла;
    
    JSONФайлов = Driver.СписокФайлов(Путь);
    
    Если Driver.FileCount = 0 Тогда
        Сообщить(Префикс + "[Пустая папка] " + Путь);
        Возврат;
    КонецЕсли;
    
    ЧтениеJSON = Новый ЧтениеJSON;
    ЧтениеJSON.УстановитьСтроку(JSONФайлов);
    МассивОбъектов = ПрочитатьJSON(ЧтениеJSON);
    ЧтениеJSON.Закрыть();
    
    Для каждого Объект Из МассивОбъектов Цикл
        Если Объект.IsFolder = Истина Тогда
            Сообщить(Префикс + "[Папка] " + Объект.Name);
            // Рекурсивный вызов для вложенной папки
            НовыйПуть = Если(Путь = "", "/" + Объект.Name, Путь + "/" + Объект.Name);
            ОбойтиПапку(НовыйПуть, Глубина + 1);
        Иначе
            РазмерКб = Округ(Объект.Size / 1024, 1);
            Сообщить(Префикс + "[Файл] " + Объект.Name + " (" + РазмерКб + " КБ)");
        КонецЕсли;
    КонецЦикла;
КонецФункции

// Вызов рекурсивного обхода с корня
ОбойтиПапку("", 0);
```

### Шаг 8: Фильтрация файлов по типу

```bsl
// Получаем только файлы изображений
JSONФайлов = Driver.СписокФайлов("/InternalStorage/DCIM/Camera");

ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.УстановитьСтроку(JSONФайлов);
МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
ЧтениеJSON.Закрыть();

КоличествоИзображений = 0;

Для каждого Файл Из МассивФайлов Цикл
    // Фильтруем только изображения
    Если Найти(НижнийРегистр(Файл.MimeType), "image/") > 0 Тогда
        КоличествоИзображений = КоличествоИзображений + 1;
        Сообщить("Изображение #" + Строка(КоличествоИзображений) + ": " + Файл.Name);
        
        // Можно добавить сохранение информации о файле
        // Например, в таблицу значений
    КонецЕсли;
КонецЦикла;

Сообщить("Найдено изображений: " + КоличествоИзображений);
```

### Шаг 9: Отключение от устройства

```bsl
// Отключаемся от устройства
Если Driver.Отключить() = Истина Тогда
    Сообщить("Отключено от устройства");
Иначе
    Сообщить("Ошибка при отключении: " + Driver.Status);
КонецЕсли;
```

## Полный пример в одном блоке

```bsl
// =====================================================================
// Полный пример работы с ADBFileDriver
// =====================================================================

// 1. Создаем компоненту
Driver = New('AddIn.ADBFileDriver.ADBFileDriver');

// 2. (Опционально) Настраиваем логирование для отладки:
// Driver.EnableLog = Истина;
// Driver.LogPath = "%TEMP%\ADBFileDriver.log";

// 3. Перечисляем устройства
JSONУстройств = Driver.ПеречислитьУстройства();
ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.УстановитьСтроку(JSONУстройств);
МассивУстройств = ПрочитатьJSON(ЧтениеJSON);
ЧтениеJSON.Закрыть();

Если МассивУстройств.Количество() = 0 Тогда
    Сообщить("MTP устройства не подключены!");
    Возврат;
КонецЕсли;

Сообщить("Найдено устройств: " + Driver.DeviceCount);

// 4. Подключаемся к первому устройству
Устройство = МассивУстройств[0];
Если Driver.Подключить(Устройство.Name) = Ложь Тогда
    Сообщить("Ошибка подключения: " + Driver.Status);
    Возврат;
КонецЕсли;

Сообщить("Подключено: " + Driver.Status);

// 5. Получаем список файлов в корне
JSONФайлов = Driver.СписокФайлов("");

ЧтениеJSON.УстановитьСтроку(JSONФайлов);
МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
ЧтениеJSON.Закрыть();

Сообщить("Файлов/папок в корне: " + Driver.FileCount);

Для каждого Файл Из МассивФайлов Цикл
    Префикс = Если(Файл.IsFolder, "[DIR] ", "[FILE]");
    Сообщить(Префикс + Файл.Name);
КонецЦикла;

// 6. Получаем файлы из папки Downloads
JSONФайлов = Driver.СписокФайлов("/InternalStorage/Downloads");

ЧтениеJSON.УстановитьСтроку(JSONФайлов);
МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
ЧтениеJSON.Закрыть();

Сообщить("Файлов в Downloads: " + Driver.FileCount);

Для каждого Файл Из МассивФайлов Цикл
    Если Файл.IsFolder = Ложь Тогда
        Сообщить("  " + Файл.Name + " (" + Формат(Файл.Size, "ЧЛЦ=10; ЧГ=0") + " байт)");
    Иначе
        Сообщить("  [Папка] " + Файл.Name);
    КонецЕсли;
КонецЦикла;

// 7. Отключаемся
Driver.Отключить();
Сообщить("Готово!");
```

## Пример: Таблица значений с файлами

```bsl
// Создание ТаблицыЗначений для хранения списка файлов
ТЗФайлов = Новый ТаблицаЗначений;
ТЗФайлов.Колонки.Добавить("Имя", Новый ОписаниеТипов("Строка"), "Имя файла");
ТЗФайлов.Колонки.Добавить("Путь", Новый ОписаниеТипов("Строка"), "Путь к файлу");
ТЗФайлов.Колонки.Добавить("Размер", Новый ОписаниеТипов("Число"), "Размер в байтах");
ТЗФайлов.Колонки.Добавить("IsFolder", Новый ОписаниеТипов("Булево"), "Признак папки");
ТЗФайлов.Колонки.Добавить("MimeType", Новый ОписаниеТипов("Строка"), "MIME-тип");

// Получаем список файлов
JSONФайлов = Driver.СписокФайлов("/InternalStorage/Downloads");

ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.УстановитьСтроку(JSONФайлов);
МассивФайлов = ПрочитатьJSON(ЧтениеJSON);
ЧтениеJSON.Закрыть();

// Заполняем ТаблицуЗначений
Для каждого Файл Из МассивФайлов Цикл
    НоваяСтрока = ТЗФайлов.Добавить();
    НоваяСтрока.Имя = Файл.Name;
    НоваяСтрока.Путь = "/InternalStorage/Downloads/" + Файл.Name;
    НоваяСтрока.Размер = Файл.Size;
    НоваяСтрока.IsFolder = Файл.IsFolder;
    НоваяСтрока.MimeType = Файл.MimeType;
КонецЦикла;

// Теперь ТЗФайлов можно использовать для отображения в форме
// Например, передать в КомпонентТаблицы
```

## Возможные пути для СписокФайлов

| Путь | Описание |
|------|----------|
| `""` или не передан | Корень устройства |
| `"/InternalStorage"` | Внутреннее хранилище |
| `"/InternalStorage/Downloads"` | Папка загрузок |
| `"/InternalStorage/DCIM"` | Фото и видео (DCIM) |
| `"/InternalStorage/DCIM/Camera"` | Снятые фото |
| `"/InternalStorage/Documents"` | Документы |
| `"/InternalStorage/Pictures"` | Изображения |
| `"/InternalStorage/Music"` | Музыка |
| `"/InternalStorage/Alarms"` | Сигналы будильника |
| `"/InternalStorage/Notifications"` | Уведомления |
| `"/InternalStorage/Ringtones"` | Рингтоны |
| `"/InternalStorage/Android/data/"` | Данные приложений |

## Обработка ошибок

```bsl
Попытка
    JSONФайлов = Driver.СписокФайлов("/InternalStorage/Downloads");
    
    Если Driver.FileCount = 0 Тогда
        Сообщить("Папка пуста или не существует");
    Иначе
        Сообщить("Файлов найдено: " + Driver.FileCount);
    КонецЕсли;
    
Исключение
    Сообщить("Ошибка при получении списка файлов:");
    Сообщить(ОписаниеОшибки());
КонецПопытка;
```

## Важные примечания

1. **Перед вызовом СписокФайлов необходимо подключить устройство** методом Подключить().
2. **DeviceCount обновляется** после вызова ПеречислитьУстройства().
3. **FileCount обновляется** после вызова СписокФайлов() — содержит количество элементов в JSON.
4. **Путь указывается** от корня устройства (подобно файловой системе Unix).
5. **Результат всегда в JSON** — необходимо использовать ЧтениеJSON и ПрочитатьJSON для парсинга.
6. **IsFolder = true** означает, что объект является папкой (Size = 0).
7. **IsFolder = false** означает, что объект является файлом (Size > 0).
