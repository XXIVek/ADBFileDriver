# ADBFileDriver - Внешняя компонента для работы с Android устройствами через ADB

## Версия: 2.10.0.0

## Назначение
Компонента предназначена для работы с Android устройствами, подключенными к компьютеру через USB, с использованием ADB (Android Debug Bridge). Позволяет перечислять подключенные устройства, подключаться к ним и управлять файлами на устройстве.

## Регистрация
```bsl
Driver = New('AddIn.ADBFileDriver.ADBFileDriver')
```

## Свойства

### Version (Версия)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Возвращает версию компоненты
- **Пример:** `"2.10.0.0"`

### EnableLog (ВключитьЛогирование)
- **Тип:** Булево
- **Чтение:** Да
- **Запись:** Да
- **Описание:** Включает/выключает логирование в файл
- **Значение по умолчанию:** `Ложь`

### LogPath (ПутьДляФайлаЛогирования)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Да
- **Описание:** Путь к файлу логирования (поддерживает переменные окружения, например `%TEMP%\ADBFileDriver.log`)
- **Значение по умолчанию:** `%TEMP%\ADBFileDriver.log`

### Status (Статус)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Текущий статус работы компоненты
- **Возможные значения:** `"Не подключен"`, `"Подключено (ADB)"`, `"Ошибка: ..."`

### DeviceCount (КоличествоУстройств)
- **Тип:** Число
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Количество подключенных ADB устройств (обновляется после вызова ПеречислитьУстройства)

### FileCount (КоличествоФайлов)
- **Тип:** Число
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Количество файлов в последнем результате вызова СписокФайлов (обновляется после вызова соответствующего метода)

### ТекущийКаталог
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Возвращает путь текущего каталога (обновляется после вызова СписокФайлов)
- **Пример:** `""` (корень), `"/DCIM"`, `"/Download"`

### КаталогDLL
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Да
- **Описание:** Каталог расположения DLL компонента. Используется для поиска `adb.exe` относительно расположения DLL.
- **Поведение при чтении:** Возвращает полный путь к каталогу, где расположена загруженная DLL (заканчивается обратным слэшем `\`)
- **Поведение при записи:** Явно задаёт каталог для поиска файлов. При записи автоматически добавляется обратный слэш в конец пути. Если путь заканчивается именем файла, он отрезается, остаётся только каталог.
- **Пример:** `"C:\TMP\ADBFileDriver\"`

## Методы

### ПеречислитьУстройства()
- **Назначение:** Возвращает список подключенных ADB устройств
- **Параметры:** Нет
- **Возвращаемое значение:** Строка в формате JSON
- **Формат JSON:**
```json
[
  {
    "Serial": "ABCDEF123456",
    "Name": "Redmi 9C NFC"
  }
]
```
- **Поля объекта:**
  - `Serial` — ADB Serial устройства
  - `Name` — имя устройства (Friendly Name)
- **DeviceCount** обновляется автоматически после вызова
- **Пример использования в 1С:**
```bsl
Driver.ПеречислитьУстройства()
// Возвращает: [{"Serial":"ABCDEF123456","Name":"Redmi 9C NFC"},...]
Сообщить("Найдено устройств: " + Driver.DeviceCount)
```

### Подключить(ИмяУстройства)
- **Назначение:** Подключается к указанному ADB устройству
- **Параметры:**
  - `ИмяУстройства` (необязательный, Строка) - имя устройства из списка ПеречислитьУстройства
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Поведение:**
  - Если параметр не передан или пустой — подключается к первому найденному ADB устройству
  - Если параметр передан — ищет устройство по имени в результате ПеречислитьУстройства()
- **Пример использования в 1С:**
```bsl
// Подключение к первому устройству
Если Driver.Подключить() = Истина Тогда
    Message("Подключено! Статус: " + Driver.Status)
КонецЕсли;

// Подключение к конкретному устройству
Если Driver.Подключить("Redmi 9C NFC") = Истина Тогда
    Message("Подключено!")
Иначе
    Message("Ошибка подключения: " + Driver.Status)
КонецЕсли;
```

### Отключить()
- **Назначение:** Отключается от подключенного устройства
- **Параметры:** Нет
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Пример использования в 1С:**
```bsl
Driver.Отключить()
```

### СписокФайлов(Путь)
- **Назначение:** Возвращает список файлов и папок на подключенном ADB устройстве
- **Параметры:**
  - `Путь` (необязательный, Строка) — относительный путь от корня устройства
    - Пустая строка или не передан — корневая директория устройства
- **Возвращаемое значение:** Строка в формате JSON
- **Формат JSON:**
```json
[
  {"Name": "DCIM", "IsFolder": true},
  {"Name": "Download", "IsFolder": true},
  {"Name": "photo.jpg", "IsFolder": false},
  {"Name": "video.mp4", "IsFolder": false}
]
```
- **Поля объекта:**
  - `Name` — имя файла или папки
  - `IsFolder` — `true` если объект является папкой, `false` если файл
- **FileCount** обновляется автоматически после вызова
- **ТекущийКаталог** обновляется автоматически после вызова
- **Пример использования в 1С:**
```bsl
// Получить список файлов в корне устройства
Список = Driver.СписокФайлов("")

// Получить список файлов в папке DCIM
Список = Driver.СписокФайлов("DCIM")

// Обработка результата
Если Driver.FileCount > 0 Тогда
    Message("Найдено объектов: " + Driver.FileCount)
    Message("Текущий каталог: " + Driver.ТекущийКаталог)
КонецЕсли;
```

### СкачатьФайл(ИмяФайла)
- **Назначение:** Скачивает **содержимое** файла с ADB устройства
- **Параметры:**
  - `ИмяФайла` (обязательный, Строка) — имя файла на устройстве (относительно ТекущийКаталог)
- **Возвращаемое значение:** Строка — содержимое файла
- **Пример использования в 1С:**
```bsl
Содержимое = Driver.СкачатьФайл("config.txt")
Если Содержимое <> "" Тогда
    Message("Содержимое: " + Содержимое)
Иначе
    Message("Файл пустой или ошибка")
КонецЕсли;
```

### ЗагрузитьФайл(ИмяНаУстройстве, Содержание)
- **Назначение:** Загружает содержимое файла на ADB устройство
- **Параметры:**
  - `ИмяНаУстройстве` (обязательный, Строка) — имя файла на устройстве (относительно ТекущийКаталог)
  - `Содержание` (обязательный, Строка) — содержимое файла (текст или JSON)
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Пример использования в 1С:**
```bsl
// Загрузка текстового содержимого
Если Driver.ЗагрузитьФайл("config.txt", "true,УТ11,3.0.1") = Истина Тогда
    Message("Файл загружен")
КонецЕсли;

// Загрузка JSON
JSONСодержимое = "[{""Id"":1,""Name"":""test""}]"
Если Driver.ЗагрузитьФайл("data.json", JSONСодержимое) = Истина Тогда
    Message("JSON файл загружен")
КонецЕсли;
```

### УдалитьФайл(ИмяФайла)
- **Назначение:** Удаляет файл на ADB устройстве
- **Параметры:**
  - `ИмяФайла` (обязательный, Строка) — имя файла на устройстве (относительно ТекущийКаталог)
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Пример использования в 1С:**
```bsl
Если Driver.УдалитьФайл("old_file.txt") = Истина Тогда
    Message("Файл удален")
Иначе
    Message("Ошибка удаления")
КонецЕсли;
```

### СписокИменФайлов(Путь)
- **Назначение:** Возвращает список имен файлов и папок на ADB устройстве (только Name + IsFolder)
- **Параметры:**
  - `Путь` (необязательный, Строка) — относительный путь от корня устройства
- **Возвращаемое значение:** Строка в формате JSON
- **Формат JSON:**
```json
[
  {"Name": "DCIM", "IsFolder": true},
  {"Name": "Download", "IsFolder": true},
  {"Name": "photo.jpg", "IsFolder": false},
  {"Name": "video.mp4", "IsFolder": false}
]
```
- **Пример использования в 1С:**
```bsl
// Получить только имена файлов в корне устройства
СписокИмен = Driver.СписокИменФайлов("")

// Получить только имена файлов в папке DCIM
СписокИмен = Driver.СписокИменФайлов("DCIM")

// Обработка результата
Если Driver.FileCount > 0 Тогда
    Message("Найдено объектов: " + Driver.FileCount)
КонецЕсли;
```

### СписокФайловРекурсивно(Путь)
- **Назначение:** Возвращает список файлов и папок на ADB устройстве (рекурсивный обход)
- **Параметры:**
  - `Путь` (необязательный, Строка) — относительный путь от корня устройства
- **Возвращаемое значение:** Строка в формате JSON
- **Формат JSON:** Аналогично СписокФайлов()
- **Пример использования в 1С:**
```bsl
Список = Driver.СписокФайловРекурсивно("")
```

## Пример полного использования в 1С

```bsl
// =============================================
// Пример полного сценария работы с ADBFileDriver
// =============================================

// 1. Подключение компоненты
ПутьКФайлуDLL = "C:\TMP\ADBFileDriver\1C\Release\Win64\ADBFileDriver_Win64.dll";
Результат = ПодключитьВнешнююКомпоненту(ПутьКФайлуDLL, "ADBFileDriver", ТипВнешнейКомпоненты.Native);

Если Не Результат Тогда
    Сообщить("Не удалось подключить компоненту");
    Возврат;
КонецЕсли;

// 2. Создание объекта
Драйвер = Новый("AddIn.ADBFileDriver.ADBFileDriver");

// 3. Настройка
Драйвер.КаталогDLL = "C:\TMP\ADBFileDriver\";
Драйвер.EnableLog = Истина;
Драйвер.LogPath = "%TEMP%\ADBFileDriver.log";

// 4. Чтение свойств
Сообщить("Версия: " + Драйвер.Version);
Сообщить("Статус: " + Драйвер.Status);

// 5. Перечисление устройств
СписокУстройств = Драйвер.ПеречислитьУстройства();
Сообщить("Найдено устройств: " + Драйвер.DeviceCount);

Если Драйвер.DeviceCount = 0 Тогда
    Сообщить("Устройства не найдены");
    Возврат;
КонецЕсли;

// 6. Подключение (к первому устройству)
Результат = Драйвер.Подключить();
Если Не Результат Тогда
    Сообщить("Ошибка подключения: " + Драйвер.Status);
    Возврат;
КонецЕсли;

Сообщить("Подключено: " + Драйвер.Status);

// 7. Получение списка файлов
СписокФайлов = Драйвер.СписокФайлов("");
Сообщить("Объектов в корне: " + Драйвер.FileCount);
Сообщить("Текущий каталог: " + Драйвер.ТекущийКаталог);

// 8. Скачивание файла
Содержимое = Драйвер.СкачатьФайл("config.txt");
Сообщить("Содержимое config.txt: " + Содержимое);

// 9. Загрузка файла
Результат = Драйвер.ЗагрузитьФайл("new_config.txt", "true,УТ11,3.0.1");
Если Результат Тогда
    Сообщить("Файл new_config.txt загружен");
КонецЕсли;

// 10. Удаление файла
Результат = Драйвер.УдалитьФайл("old_file.txt");
Если Результат Тогда
    Сообщить("Файл old_file.txt удален");
КонецЕсли;

// 11. Отключение
Драйвер.Отключить();
Сообщить("Отключено: " + Драйвер.Status);
```

## Файл логирования
- **Путь по умолчанию:** `%TEMP%\ADBFileDriver.log`
- **Формат записи:** `[ЧЧ:ММ:СС.мсс] Сообщение`
- **Пример:** `[18:45:23.456] EnumerateMtpDevices: Начинаем перечисление MTP устройств`

## Требования
- Windows 10 или выше
- Поддерживаются Android устройства с включенным USB-отладкой
- Устройства должны быть подключены через USB
- ADB драйверы должны быть установлены
