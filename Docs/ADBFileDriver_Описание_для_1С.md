# ADBFileDriver - Внешняя компонента для работы с MTP устройствами

## Назначение
Компонента предназначена для работы с MTP (Media Transfer Protocol) устройствами, подключенными к компьютеру через USB. Позволяет перечислять подключенные устройства, подключаться к ним и управлять файлами на устройстве.

## Регистрация
```bsl
Driver = New('AddIn.ADBFileDriver.ADBFileDriver')
```

## Свойства

### Version (Версия)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Возвращает версию компоненты
- **Пример:** `"2.0.0.0"`

### EnableLog (ВключитьЛогирование)
- **Тип:** Булево
- **Чтение:** Да
- **Запись:** Да
- **Описание:** Включает/выключает логирование в файл
- **Значение по умолчанию:** `Ложь`

### LogPath (ПутьДляФайлаЛогирования)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Да
- **Описание:** Путь к файлу логирования (поддерживает переменные окружения, например `%TEMP%\ADBFileDriver.log`)
- **Значение по умолчанию:** `%TEMP%\ADBFileDriver.log`

### Status (Статус)
- **Тип:** Строка
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Текущий статус работы компоненты
- **Возможные значения:** `"Не подключен"`, `"Подключено"`, `"Ошибка: ..."`

### DeviceCount (КоличествоУстройств)
- **Тип:** Число
- **Чтение:** Да
- **Запись:** Нет
- **Описание:** Количество подключенных MTP устройств (обновляется после вызова ПеречислитьУстройства)

## Методы

### ПеречислитьУстройства()
- **Назначение:** Возвращает список подключенных MTP устройств
- **Параметры:** Нет
- **Возвращаемое значение:** Строка в формате JSON
- **Формат JSON:**
```json
[
  {"Id": "{устройство-ID}", "Name": "ИмяУстройства"},
  {"Id": "{устройство-ID}", "Name": "ИмяУстройства"}
]
```
- **Пример использования в 1С:**
```bsl
Driver.ПеречислитьУстройства()
// Возвращает: [{"Id":"", "Name":"Redmi 9C NFC"},...]
```

### Подключить(ИмяУстройства)
- **Назначение:** Подключается к указанному MTP устройству
- **Параметры:**
  - `ИмяУстройства` (обязательный, Строка) - имя устройства из списка ПеречислитьУстройства
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Пример использования в 1С:**
```bsl
Если Driver.Подключить("Redmi 9C NFC") = Истина Тогда
    Message("Подключено!")
Иначе
    Message("Ошибка подключения")
КонецЕсли;
```

### Отключить()
- **Назначение:** Отключается от подключенного устройства
- **Параметры:** Нет
- **Возвращаемое значение:** Булево (`Истина` при успехе, `Ложь` при ошибке)
- **Пример использования в 1С:**
```bsl
Driver.Отключить()
```

## Пример полного использования в 1С

```bsl
// Создаем компоненту
Driver = New('AddIn.ADBFileDriver.ADBFileDriver')

// Включаем логирование
Driver.EnableLog = Истина
Driver.LogPath = "%TEMP%\ADBFileDriver.log"

// Получаем список устройств
Список = Driver.ПеречислитьУстройства()
Message("Найдено устройств: " + Driver.DeviceCount)

// Подключаемся к первому найденному устройству
Если Driver.Подключить("Redmi 9C NFC") = Истина Тогда
    Message("Статус: " + Driver.Status)
    
    // ... здесь будут методы работы с файлами (Этап 2)
    
    // Отключаемся
    Driver.Отключить()
КонецЕсли;
```

## Файл логирования
- **Путь по умолчанию:** `%TEMP%\ADBFileDriver.log`
- **Формат записи:** `[ЧЧ:ММ:СС.мсс] Сообщение`
- **Пример:** `[18:45:23.456] EnumerateMtpDevices: Начинаем перечисление MTP устройств`

## Требования
- Windows 10 или выше
- Поддерживаются MTP устройства (смартфоны, планшеты)
- Устройства должны быть в режиме "Передача файлов" / MTP
